# Configuração de Agentes — Frontend Geral

## 📋 Regras Gerais do Projeto

### Idioma e Convenções
- Código, classes, variáveis, propriedades e arquivos em **inglês**.
- Comentários, documentação e commits em **português**.
- Commits semânticos: `feat:`, `fix:`, `docs:`, `refactor:`, `test:`, `chore:`.

### Stack Tecnológica
- HTML5, CSS3, JavaScript/TypeScript
- Frameworks modernos (React, Vue, Angular, Svelte) ou Vanilla JS
- Tailwind CSS / CSS Modules / Sass
- Linters (ESLint, Biome) e Formatadores (Prettier)

---

## 🏗️ Padrões de Projeto (Frontend Best Practices)
- **Componentização**: Mantenha componentes pequenos, reutilizáveis e focados em uma única responsabilidade.
- **Responsividade**: Siga a abordagem mobile-first. Teste em múltiplos breakpoints.
- **Acessibilidade (a11y)**: Use elementos HTML semânticos, forneça `alt` para imagens, use `aria-labels` e garanta navegação por teclado adequada.
- **Gerenciamento de Estado**: Evite prop-drilling excessivo. Use contextos ou bibliotecas de estado global apropriadas.
